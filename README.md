# LuminaResc: Smart Mine Safety & Evacuation System

## 🚨 Problem Statement
Mining environments are highly dangerous due to risks like gas leaks and seismic activities. During emergencies, miners often lack clear evacuation guidance, leading to delayed response and increased casualties. Additionally, continuous lighting systems consume unnecessary energy.

## 💡 Solution
LuminaResc is an intelligent mine safety system designed to detect hazards and guide miners to safe evacuation paths using adaptive lighting. The system enhances safety while also optimizing energy consumption.

## ⚙️ Features
- 🔹 Gas detection using PPM sensors
- 🔹 Seismic activity monitoring
- 🔹 Adaptive LED guidance system for evacuation
- 🔹 Energy-efficient lighting control
- 🔹 Real-time hazard indication

## 🛠️ Tech Stack
- MATLAB (simulation & analysis)
- Sensors (Gas + Seismic)
- Embedded system logic

## 🧠 How It Works
- The system continuously monitors environmental conditions using gas and seismic sensors.
- When a hazard is detected, the system activates a guided lighting path.
- LEDs dynamically adjust brightness based on the miner’s distance.
- This ensures clear evacuation direction while minimizing energy usage.

## 📊 Results

- Achieved up to ~n% reduction in energy consumption using adaptive LED control compared to static lighting, validated through MATLAB simulations.(based on various simulations)

- Reduced evacuation decision time by guiding miners through dynamically illuminated safe paths during emergency scenarios.

- Implemented real-time hazard detection using gas concentration thresholds (PPM) and seismic activity triggers.

- Simulation results demonstrate effective route optimization and safer evacuation under multiple hazard scenarios.
