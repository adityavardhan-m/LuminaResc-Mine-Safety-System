# LuminaResc – Mine Safety System

## Problem
Miners face delayed evacuation during hazards like gas leaks and seismic activity.

## Solution
A smart LED guidance system that dynamically adjusts brightness based on miner distance, creating a clear evacuation path.

## Features
- Distance-based LED brightness control
- Gas & seismic hazard detection
- Energy-efficient lighting system

## Results
- Reduced evacuation decision time (simulated)
- Improved visibility in low-light conditions
- Energy savings compared to static lighting
